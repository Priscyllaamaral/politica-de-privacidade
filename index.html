Política de Privacidade

Última atualização: 27 de novembro de 2025

Este aplicativo foi desenvolvido com o objetivo de oferecer conteúdo e funcionalidades de forma simples e segura para todos os usuários.

1. Coleta de Dados

Este aplicativo não coleta, armazena ou compartilha quaisquer dados pessoais dos usuários. Nenhuma informação é solicitada, registrada ou enviada para servidores externos.

2. Permissões

O aplicativo não solicita permissões sensíveis, como acesso à câmera, localização, contatos ou arquivos do dispositivo.

3. Uso de Informações

Como não são coletados dados, nenhuma informação é utilizada para fins de análise, publicidade ou rastreamento.

4. Serviços de Terceiros

O aplicativo não integra serviços de terceiros que coletem informações automaticamente.

5. Segurança

Como nenhuma informação do usuário é coletada ou transmitida, não há risco de exposição de dados pessoais.

6. Alterações nesta Política

Caso no futuro sejam adicionadas funcionalidades que exijam coleta ou uso de dados, esta política será atualizada e os usuários serão informados por meio de atualização do aplicativo e por meio desta página.
